# FresherFit AI

Auto-generated by ETS-AI 10x.

## Idea Summary
An AI-powered resume screener for Indian fresher software engineers to help them apply for jobs.

## Problem Statement
Indian fresher software engineers struggle to effectively tailor their resumes to job descriptions and identify missing skills when applying for jobs.

## Target Persona
Indian fresher software engineer

## MVP Features
- User registration and profile management
- Resume upload (PDF, DOCX) and parsing
- Job Description (JD) input (text paste or upload)
- AI-powered skill gap analysis between resume and JD
- Tailoring suggestions for resume based on JD (keywords to add/emphasize)
- Resume-job fit score
- Dashboard to view analysis history
- Basic subscription management for premium features

## Tech Stack
Modern web stack: Python (Django/FastAPI) for backend, React/Vue for frontend, PostgreSQL for database, AWS/GCP for cloud infrastructure, with ML services (e.g., Hugging Face, custom NLP models) for AI components.
